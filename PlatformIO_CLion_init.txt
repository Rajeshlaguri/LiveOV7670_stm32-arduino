
Init CLion project with PlatformIO:

1. Install PlatformIO command line tools:
http://docs.platformio.org/en/stable/installation.html

Install PlatformIO with Python package manager:
sudo pip install -U platformio


2. cd to project root


3. run (board type should change if different CPU version is used)
platformio init -d "./stm32-arduino" --ide clion --board bluepill_f103c8 --project-option "framework=arduino" --project-option "upload_protocol=serial" --project-option "build_flags=-std=gnu++11"


4. "stm32/.idea" folder generated by "platformio init" can be deleted.
CLion will create its own project file in the root directory
